Installation

Initialize your project, add components, and start building.

1. Initialize your project

# Create components.json and base CSS
npx uilibrary init

2. Add components you need

# Add individual components
npx uilibrary add button
npx uilibrary add modal tabs accordion

# See all available components
npx uilibrary list

3. Build your CSS

npx @tailwindcss/cli -i src/base.css -o dist/output.css

4. Use in your HTML

<!-- Add compiled CSS -->
<link rel="stylesheet" href="dist/output.css">

<!-- Add Stimulus -->
<script src="https://cdn.jsdelivr.net/npm/@hotwired/stimulus/dist/stimulus.umd.js"></script>

<!-- Use a component -->
<button data-controller="button" data-button-variant-value="default">
  Click me
</button>
